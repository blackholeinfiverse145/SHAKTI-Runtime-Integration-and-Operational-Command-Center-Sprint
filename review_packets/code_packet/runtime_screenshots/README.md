# Runtime Screenshots

This directory serves as the pointer for dashboard runtime screenshots.

## Screenshot Storage Pointer
To avoid duplicating large binary image assets and causing repository bloat, all visual evidence screenshots are stored in the root-level [`evidence/`](file:///c:/Pratik_Bhuwad/shakti-command-center/evidence) folder of this repository.

Reviewers should navigate to that directory to inspect full-size captures, including:
*   `01_dashboard_startup.png`: Initial page layout loads.
*   `04_operations_compute.png` to `13_evidence_intelligence.png`: Close-up layout renders of all 10 dashboard zones.
*   `17_loading_state.png` & `18_empty_state.png`: Skeletons and fallback states.
*   `19_error_recovery.png` & `20_offline_banner.png`: Connection lost states and recovery controls.
*   `24_dark_theme.png`: Visual layout of the dashboard dark theme.
*   `26_typography_hierarchy.png`: Visual overview of typography scaling and weights.

Detailed descriptions of what each file proves are mapped inside the [`Evidence_Guide.md`](file:///c:/Pratik_Bhuwad/shakti-command-center/review_packets/code_packet/Evidence_Guide.md) file.
